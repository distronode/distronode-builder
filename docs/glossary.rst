Glossary
========


.. glossary::

    execution environment
      execution environments are container images intended to be used as Distronode control nodes.
      Starting in version 2.0, `distronode-runner <https://distronode-runner.readthedocs.io/en/latest/>`_.
      can make use of these images.


    Control node
      The machine or container running Distronode.
      See the `Distronode documentation <https://docs.distronode.com/distronode/latest/network/getting_started/basic_concepts.html#control-node>`_
      for a more comprehensive explanation.
